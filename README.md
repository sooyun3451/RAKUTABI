# 🏨 라쿠타비 (RAKUTABI)

일본 숙소 예약 사이트  
사용자는 원하는 숙소를 검색하고 예약할 수 있습니다.

## ✨ 주요 기능

- 숙소 목록 조회 및 상세 보기
- 숙소 예약 및 예약 관리
- 사용자 회원가입 / 로그인
- 사용자 마이페이지
- 식당 리뷰 목록 조회 및 상세 보기

---

## 🛠 기술 스택

- **Frontend:** React, Axios, React Router
- **기타:** CSS, HTML

---

## ⚙️ 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/username/rakutabi.git

# 패키지 설치
npm install

# 개발 서버 실행
npm run dev

---
```
